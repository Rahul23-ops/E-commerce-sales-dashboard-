# E-commerce-sales-dashboard-
E-Commerce Sales Dashboard developed using Power BI.
This dashboard provides a comprehensive view of sales performance, profit trends, customer purchasing behavior, product category performance, and payment preferences. The objective of this project was to transform raw sales data into meaningful insights that can support business decision-making.
Key Highlights:
🔹 Total Sales: 438K
 🔹 Total Quantity Sold: 6K
 🔹 Total Profit: 37K
 🔹 State-wise Sales Analysis
 🔹 Category-wise Sales Distribution
 🔹 Monthly Profit Trend Analysis
 🔹 Customer-wise Sales Performance
 🔹 Payment Mode Analysis
 🔹 Sub-Category Profitability Insights
Tools Used:
✅ Power BI
 ✅ Power Query
 ✅ DAX
 ✅ Excel
